# 🌿 NutriFlex — Landing Page

> Official landing page for the **NutriFlex** Android app — a smart nutrition and meal planning platform built by the Nutrition Navigators team.

---

## ✨ Overview

This is a fully responsive, modern landing page designed to showcase the NutriFlex mobile application. Built with a **dark luxury aesthetic**, smooth animations, and a premium UI — crafted entirely with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies.

---

## 🚀 Features

- **Animated Splash Screen** — Logo reveal with pulsing rings and a loading bar
- **Sticky Navbar** — Scroll-aware with blur effect, active link tracking and mobile drawer
- **Hero Section** — Floating phone mockup, star canvas background and cursor parallax effect
- **Features Grid** — 6 real app features with animated icon cards
- **How It Works** — Vertical alternating timeline with 5 onboarding steps
- **App Screenshots** — Phone-framed screenshot gallery (2×2 grid on mobile)
- **BMI Calculator** — Full parameter input with animated gauge and macro breakdown
- **About & Team** — Mission statement with a premium 2-column team card grid
- **Contact Section** — Contact info cards and a working contact form
- **Footer** — Clean minimal 3-column layout
